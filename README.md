# BlockDataSaver
A Minecraft Spigot Plugin to save block NBT

一个Minecraft插件可以保存方块NBT数据，破坏方块时掉落带有NBT的方块。

A Minecraft Spigot Plugin to save block NBT，drop item with NBT when breaking that block.
